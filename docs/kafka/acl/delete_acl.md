# Delete Kafka ACLs

To delete an ACL in AxonOps:

1. Go to `ACLS` in the left navigation.
2. Use the search field to locate the ACL by topic, principal, or operation.
3. Select the ACL you want to remove and click `Remove ACL`.
4. Confirm the deletion.

!!! warning
    Deleting an ACL takes effect immediately. Ensure the ACL is not required by production clients before removal.
