# Set up SMTP / Email Notifications

On the AxonOps application menu, select `Settings -> Integrations`.

Click on the `SMTP` area.


[![integrations](imgs/integrations.jpg)](imgs/integrations.jpg)

Fill in the required SMTP details:

- SMTP host and port
- TLS/SSL settings
- Username and password (if required)
- From address
