# Set up ServiceNow Integration

## Log in to ServiceNow

Navigate to `Settings -> Integrations` and click `ServiceNow`.

![servicenow-1](imgs/servicenow-1.png)

Gather the `instance name`, `username`, and `password` from ServiceNow to fill out the form:
![servicenow-2](imgs/servicenow-2.png)

## Update ServiceNow Incident Template

If you want to see the detailed description of a notification, add the `description` field to the ServiceNow incident templates.

![servicenow-3](imgs/servicenow-3.png)
